======================
The ``geomics`` module
======================

This document describe the ``geomics`` module.

Getting started
===============

To activate the module on ``flux`` run::
    
    module load geomicro/geomics

On the ``vondamm`` or ``cayman`` servers run::

    module load geomics

===============

This part provides information of the available programms and tools of
